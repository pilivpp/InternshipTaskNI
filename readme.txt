This program prints the date range for the entered dates.

Please read the output commands.
Remember to input dates with correct format.

The program prints three different range formats.

1) For dates with the same  month and year (example: 01.01.2020 02.01.2020) 
Output prints: 01.01-02.01.2020

2) For dates with the same year but different month (example: 01.01.2020 02.02.2020)
Output prints: 01.01 - 02.02.2020

3) For dates with different year (example: 01.01.2020 02.02.2021)
Output prints: 01.01.2020 - 02.02.2021

If you pass invalid format date, input once again the date. Program will be asking you of correct date
to the moment when you pass valid format.
